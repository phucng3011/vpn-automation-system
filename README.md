# VPN Automation & Monitoring System

## 📌 Overview
A Python-based automation system that simulates VPN monitoring, detects tunnel failures, and automatically triggers recovery actions via SSH.

## 🚀 Features
- VPN tunnel health check using ICMP
- Automated VPN restart via SSH (Paramiko)
- Real-time logging and alerting
- Configurable multi-site architecture

## 🛠 Tech Stack
- Python 3
- Paramiko (SSH automation)
- YAML config
